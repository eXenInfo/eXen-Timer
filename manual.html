Einführung

Herzlich willkommen zum eXen-Timer! Dieser Timer wurde speziell für die Anforderungen der dynamischen Schießdisziplinen des BDMP entwickelt. Er hilft Ihnen, Trainingsabläufe präzise zu steuern, eigene Disziplinen zu erstellen und diese einfach mit anderen zu teilen.

Die App besteht aus drei Hauptbereichen: Timer, Admin und Hilfe.

Die Timer-Ansicht

Dies ist Ihre Kommandozentrale am Schießstand. Die Ansicht ist bewusst minimalistisch gehalten, um eine schnelle und fehlerfreie Bedienung zu gewährleisten.

1. Eine Disziplin durchführen

  1. Disziplin laden: Wechseln Sie zuerst in die "Admin"-Ansicht. Wählen Sie eine Disziplin aus der Liste aus und klicken Sie auf den Laden-Knopf. Die App wechselt automatisch zurück in die Timer-Ansicht.
  2. Ablauf starten: Klicken Sie auf den großen grünen Start-Knopf, um die erste Phase zu beginnen.
  3. Pausen & Nächste Phase: Wenn eine Phase die Option "Pause nachher" aktiviert hat, hält der Timer an und zeigt einen Weiter-Knopf. Ein Klick darauf bereitet die nächste Phase vor, die Sie dann wieder mit "Start" auslösen.
  4. Reset: Ein Klick auf den roten Reset-Knopf bricht den gesamten Ablauf sofort ab und setzt die Disziplin auf den Anfang der ersten Phase zurück.

2. Einzelne Phasen trainieren

Sie müssen nicht immer den gesamten Ablauf starten. In der Liste der Phasen unter dem Timer finden Sie neben jeder Phase ein Play-Symbol (▶). Ein Klick darauf setzt den Timer direkt an den Anfang dieser spezifischen Phase. So können Sie gezielt einzelne Übungen wiederholen.

Die Admin-Ansicht

Hier verwalten Sie Ihre Inhalte. Sie können Disziplinen bearbeiten, neue erstellen, importieren und exportieren.

1. Disziplin-Verwaltung

  • Laden: Überträgt die ausgewählte Disziplin in die Timer-Ansicht und macht sie startklar.
  • Kopieren: Kopiert die Daten der ausgewählten Disziplin als Text (im JSON-Format) in Ihre Zwischenablage.
  • Löschen: Entfernt die ausgewählte Disziplin dauerhaft aus dem Speicher Ihres Geräts.

2. Sammlung Teilen / Sichern

  • Sammlung kopieren: Kopiert **alle** Ihre gespeicherten Disziplinen in die Zwischenablage. Ideal für ein komplettes Backup.
  • Aus Text einfügen: Öffnet ein Fenster, in das Sie zuvor kopierte Disziplin-Daten einfügen können, um sie zu importieren.

3. Standard-Satz laden

Ein Klick auf BDMP-Standard laden lädt eine von uns kuratierte Liste von Standard-Disziplinen direkt vom Server in Ihre App. Bestehende Disziplinen mit gleichem Namen werden dabei überschrieben.

4. Der Disziplin-Editor: Eigene Abläufe erstellen

Hier liegt das Herzstück der App. Wenn Sie eine Disziplin aus der Liste auswählen, wird sie im Editor geladen. Sie können aber auch eine komplett neue erstellen.

Phasen bearbeiten & verwalten

  • Bearbeiten: Klicken Sie direkt auf den Text einer Phase, um sie im "Phase bearbeiten"-Fenster anzupassen.
  • Verschieben: Nutzen Sie die Pfeil-Knöpfe (▲/▼) rechts, um die Reihenfolge der Phasen zu ändern.
  • Duplizieren (❐): Erstellt eine Kopie einer Phase direkt darunter.
  • Löschen (❌): Entfernt eine Phase.

Bedeutung der Felder für eine Phase

  • Beschreibung: Der Name oder die Anweisung für die Phase. Zeilenumbrüche werden im Timer korrekt dargestellt.
  • Vorlaufzeit (s): Eine Vorbereitungszeit in Sekunden, bevor die eigentliche Dauer startet (z. B. für das Kommando "Fertig?").
  • Dauer (s): Die Hauptzeit der Übung in Sekunden.
  • Wiederholungen: Wie oft diese Phase hintereinander ausgeführt werden soll.
  • Pause zw. Wdh. (s): Eine kurze Pause in Sekunden zwischen den einzelnen Wiederholungen.
  • Ton am Anfang / Ende: Legt fest, ob zu Beginn oder am Ende der Dauer (oder beidem) ein Signalton ertönen soll.
  • Pause nach Phase: Wenn aktiviert, stoppt der Timer nach dieser Phase und wartet auf eine manuelle Bestätigung ("Weiter"-Knopf).

Tipp: App auf dem Smartphone installieren

Diese Webseite ist eine Progressive Web App (PWA). Sie können sie direkt auf den Startbildschirm Ihres Geräts legen, um sie wie eine native App zu starten – inklusive Offline-Fähigkeit.

  • iPhone/iPad (Safari): Öffnen Sie die Webseite, tippen Sie auf das "Teilen"-Symbol und wählen Sie "Zum Home-Bildschirm".
  • Android (Chrome): Öffnen Sie die Webseite, tippen Sie auf das Menü (drei Punkte) und wählen Sie "App installieren" oder "Zum Startbildschirm hinzufügen".
