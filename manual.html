Deutsches Handbuch

Einführung

Herzlich willkommen zum eXen-Timer! Dieser Timer wurde entwickelt, um dir das Training und die Durchführung der dynamischen BDMP-Disziplinen zu vereinfachen. Dieses Handbuch erklärt dir alle Funktionen der App.

Die Timer-Ansicht

Dies ist deine Kommandozentrale am Schießstand, bewusst minimalistisch gehalten für eine schnelle und fehlerfreie Bedienung.

1. Eine Disziplin durchführen

  1. Disziplin laden: Wechsle in die "Admin"-Ansicht, wähle eine Disziplin und klicke auf den Laden-Knopf. Die App wechselt automatisch zurück.
  2. Ablauf starten: Klicke auf den großen grünen Start-Knopf.
  3. Pausen & Nächste Phase: Wenn eine Phase die Option "Pause nachher" hat, hält der Timer an und zeigt einen Weiter-Knopf, der die nächste Phase vorbereitet.
  4. Reset: Der rote Reset-Knopf bricht den Ablauf ab und setzt die Disziplin auf den Anfang zurück.

2. Einzelne Phasen trainieren

Klicke in der Phasen-Liste auf das Play-Symbol (▶) neben einer Übung, um direkt zu dieser zu springen und sie isoliert zu trainieren.

Die Admin-Ansicht

Hier verwaltest du deine Inhalte. Du kannst Disziplinen bearbeiten, neue erstellen, importieren und exportieren.

1. Disziplin-Verwaltung

  • Laden: Überträgt die Disziplin in die Timer-Ansicht.
  • Kopieren: Kopiert die ausgewählte Disziplin als Text in deine Zwischenablage.
  • Löschen: Entfernt die Disziplin dauerhaft aus dem Gerätespeicher.

2. Sammlung Teilen / Sichern

  • Sammlung kopieren: Kopiert **alle** deine Disziplinen als Backup.
  • Aus Text einfügen: Importiert zuvor kopierte Disziplinen.

3. Standard-Satz laden

Ein Klick auf BDMP-Standard laden lädt eine kuratierte Liste von Disziplinen vom Server. Bestehende Disziplinen werden überschrieben.

4. Der Disziplin-Editor

Hier liegt das Herzstück der App. Wähle eine Disziplin, um sie zu bearbeiten, oder erstelle eine komplett neue.

  • Bedeutung der Felder:
    • Beschreibung: Name/Anweisung für die Phase. Zeilenumbrüche (`\n`) werden korrekt dargestellt.
    • Vorlaufzeit: Vorbereitungszeit vor dem Startsignal.
    • Dauer: Hauptzeit der Übung.
    • Wiederholungen: Wie oft die Phase wiederholt wird.
    • Pause zw. Wdh.: Pause zwischen den Wiederholungen.
    • Ton am Anfang / Ende: Steuert die akustischen Signale.
    • Pause nach Phase: Erzwingt einen manuellen Stopp nach der Phase, bevor die nächste beginnt.

Tipp: App auf dem Smartphone installieren

Diese Webseite ist eine Progressive Web App (PWA). Du kannst sie auf deinem Startbildschirm ablegen, um sie wie eine normale App zu nutzen – auch offline.

User Manual (English)

Note: The application interface is in German. This manual explains the function of the German UI elements.

Introduction

Welcome to the eXen-Timer! This timer was developed to help you control and manage timings for the dynamic shooting disciplines of the German BDMP federation.

The Timer View (`Timer`)

This is your command center at the shooting range, intentionally kept minimalistic for fast and error-free operation.

1. Running a Discipline

  1. Load Discipline: First, switch to the `Admin` view. Select a discipline from the list and click the blue `Laden` (Load) button. The app will automatically switch back.
  2. Start Sequence: Click the large green `Start` button.
  3. Pauses & Next Stage: If a stage has the "Pause nachher" (Pause after) option enabled, the timer will stop and show a blue `Weiter` (Continue) button, which prepares the next stage.
  4. Reset: The red `Reset` button immediately stops the entire sequence and resets the discipline to the beginning.

2. Training Individual Stages

In the stage list below the timer, click the Play icon (▶) next to any exercise to jump directly to it for isolated practice.

The Admin View (`Admin`)

This is where you manage your content. You can edit disciplines, create new ones, and import/export them.

1. Discipline Management

  • `Laden` (Load): Transfers the selected discipline to the Timer view, making it ready to start.
  • `Kopieren` (Copy): Copies the selected discipline's data as text to your clipboard.
  • `Löschen` (Delete): Permanently removes the selected discipline from your device's storage.

2. Share / Backup Collection

  • `Sammlung kopieren` (Copy Collection): Copies **all** of your disciplines for a complete backup.
  • `Aus Text einfügen` (Paste from Text): Imports previously copied disciplines.

3. Load Default Set

Clicking BDMP-Standard laden (Load BDMP Standard) loads a curated list of disciplines from the server, overwriting existing ones with the same name.

4. The Discipline Editor

This is the core of the app. Select a discipline to edit it, or create a new one from scratch.

  • Field Descriptions:
    • `Beschreibung` (Description): Name/instruction for the stage. Line breaks (`\n`) are rendered correctly.
    • `Vorlaufzeit` (Prep Time): Preparation time before the start signal.
    • `Dauer` (Duration): The main time for the exercise.
    • `Wiederholungen` (Repetitions): How many times the stage is repeated.
    • `Pause zw. Wdh.` (Pause between Reps): A short pause between repetitions.
    • `Ton am Anfang / Ende` (Sound at Start / End): Controls the acoustic signals.
    • `Pause nach Phase` (Pause after Stage): Forces a manual stop after the stage before the next one begins.

Tip: Install the App on Your Smartphone

This website is a Progressive Web App (PWA). You can add it to your home screen to use it like a native app, including offline functionality.
